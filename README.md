# ✈️ Flight Booking System

## Overview
Backend REST API built with Spring Boot for searching flights, creating bookings, and managing reservations.

## Tech Stack
Java 17+, Spring Boot, Spring Data JPA, Hibernate, PostgreSQL/MySQL, Gradle, Lombok

## Features
- Create/List flights  
- Book/Cancel flights  
- Get booking details

## How to Run
```bash
git clone https://github.com/AbulfazGurbanli/flight-booking.git
cd flight-booking
# Configure database in src/main/resources/application.yml
./gradlew bootRun
